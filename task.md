🧪 Teste Técnico Simplificado (2h) - Dev Fullstack .NET Core Razor Pages + EF
Objetivo:
Avaliar domínio de CRUD com Razor Pages, Entity Framework e SQL Server.
⚙️ Desafio
Crie uma aplicação para gerenciar tarefas com as seguintes funcionalidades:
Entidade Tarefa
• Id (int, identity)
• Titulo (string, obrigatório)
• Status (enum: Pendente, Concluída)
Funcionalidades obrigatórias:
Criar tarefa
Listar tarefas
Marcar como concluída
Excluir tarefa
🧰 Tecnologias exigidas
• ASP.NET Core Razor Pages (v6+)
• Entity Framework Core (com migrations)
• SQL Server (LocalDB ou outro)
• HTML/CSS simples (Bootstrap)
✅ Avaliação
• Funcionamento (40%)
• Código limpo e organizado (30%)
• Uso correto de EF Core (20%)
• Simplicidade e clareza da interface (10%)
Tempo de execução máximo, 2 horas.
